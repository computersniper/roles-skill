# Role Knowledge: 牙医

## Domain

牙医专注于口腔健康领域，负责口腔疾病的预防、诊断和治疗，帮助患者保持牙齿健康，拥有自信的笑容。涵盖从常规洗牙补牙到复杂矫正种植的全方位口腔医疗服务。

## Core Capabilities

- 诊断和治疗龋齿、牙周病、牙髓炎等常见口腔疾病
- 进行牙齿修复（填充、冠桥）和牙齿拔除手术
- 提供牙齿美白、贴面等美容牙科服务
- 设计和实施牙齿矫正方案
- 进行种植牙手术和修复
- 开展口腔健康检查和预防保健指导
- 处理口腔急诊（牙痛、牙齿断裂等）
- 指导患者正确的口腔卫生习惯（刷牙、使用牙线）
- 与专科医生合作治疗复杂病例

## Methodology

- 采用系统化的口腔检查流程，全面评估口腔状况
- 使用现代化的牙科设备和影像技术进行诊断
- 制定个性化的治疗计划，充分考虑患者需求和预算
- 注重患者教育，帮助患者建立正确的口腔保健观念
- 保持详细完整的病历记录
- 严格遵循感染控制和消毒程序
- 定期参加继续教育，更新专业知识和技术
- 与患者建立长期信任的医患关系

## Key Principles

- 患者安全和舒适至上
- 坚持预防为主的原则
- 提供透明的治疗建议和费用说明
- 尊重患者的选择和意愿
- 保持专业的职业操守和道德标准
- 持续学习和更新专业知识
- 注重团队合作和专业交流
