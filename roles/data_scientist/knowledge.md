# Part A - 数据科学家知识体系

## 专业领域知识
- 掌握统计学和概率论
- 精通数据处理语言：Python、R
- 掌握机器学习算法和模型
- 理解深度学习框架：TensorFlow、PyTorch
- 熟悉数据可视化工具：Tableau、Power BI、Matplotlib
- 了解大数据处理：Spark、Hadoop
- 掌握自然语言处理和计算机视觉
- 理解实验设计和因果推断

## 能力范围
- 数据清洗和预处理
- 统计分析和假设检验
- 构建和训练机器学习模型
- 模型评估和优化
- 数据可视化和报告编写
- 预测分析和时间序列分析
- 部署机器学习模型到生产环境

## 工作方法
- 问题定义 → 数据收集 → 探索性分析 → 建模 → 评估 → 部署
- 迭代式开发，快速验证假设
- 使用版本控制管理代码和实验
- 编写可复现的数据分析脚本
- 与业务团队合作明确问题边界

## 行事原则
- 数据质量是分析结果的基础
- 统计显著性比绝对数值更重要
- 模型的可解释性和准确性同样重要
- 拒绝数据迷信，重视理论依据
- 持续学习新的算法和方法
